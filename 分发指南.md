# 课程期初资料生成工具 - 分发指南

## 一、打包应用

### 方法1：使用打包脚本（推荐）

1. **运行打包脚本**：
   ```bash
   python build_app.py
   ```

2. **打包过程**：
   - 自动安装PyInstaller（如果未安装）
   - 将应用打包成独立可执行文件
   - 创建分发包（ZIP压缩包）

3. **生成文件**：
   - `课程期初资料生成工具_分发版.zip` - 完整分发包

### 方法2：手动打包

1. **安装PyInstaller**：
   ```bash
   pip install pyinstaller
   ```

2. **执行打包命令**：
   ```bash
   pyinstaller --onefile --windowed --name=课程期初资料生成工具 course_materials_app.py
   ```

3. **手动创建分发包**：
   - 将`dist/课程期初资料生成工具.exe`复制到新目录
   - 复制`期初资料1`目录
   - 复制`使用说明.md`
   - 复制`课程信息_2026年03月17日.json`（可选，供示例使用）

## 二、分发方式

### 方式1：通过邮件发送

1. 将`课程期初资料生成工具_分发版.zip`作为附件发送
2. 在邮件中附上简要使用说明

### 方式2：通过网盘分享

1. 将ZIP包上传到百度网盘、阿里云盘等
2. 生成分享链接
3. 将链接和提取码发送给使用者

### 方式3：通过U盘拷贝

1. 将ZIP包复制到U盘中
2. 直接交给使用者

## 三、使用者操作步骤

### 步骤1：接收并解压

1. 接收分发包（ZIP压缩包）
2. 将压缩包解压到任意位置

### 步骤2：运行应用

#### Windows系统：
- 双击`课程期初资料生成工具.exe`

#### macOS系统：
- 双击`课程期初资料生成工具.app`

### 步骤3：准备工作

1. **准备JSON文件**：
   - 创建包含课程信息的JSON文件（格式见使用说明）
   - 或使用提供的示例JSON文件

2. **准备模板文件**：
   - 确保`期初资料1`目录中包含所有模板文件
   - 确保电子签名图片存在（任渝.png、陈蔚.jpg）

### 步骤4：使用应用

1. 选择JSON文件
2. 选择模板目录（默认是`期初资料1`）
3. 选择输出目录
4. 点击"生成课程资料"按钮

## 四、常见问题解决

### 问题1：应用无法运行

**可能原因**：缺少Python环境或依赖

**解决方案**：
- 使用打包好的可执行文件（已包含所有依赖）
- 确保解压完整，不要删除任何文件

### 问题2：模板文件找不到

**错误信息**："模板文件不存在"

**解决方案**：
- 确保`期初资料1`目录完整
- 检查模板文件名称是否正确

### 问题3：电子签名无法添加

**错误信息**："添加电子签名失败"

**解决方案**：
- 确保`任渝.png`和`陈蔚.jpg`存在于模板目录中
- 检查图片格式是否正确（PNG或JPG）

### 问题4：JSON文件格式错误

**错误信息**："JSON文件格式不正确"

**解决方案**：
- 检查JSON文件格式是否符合要求
- 使用提供的示例JSON文件作为参考
