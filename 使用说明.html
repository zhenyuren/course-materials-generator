课程期初资料生成工具使用说明

功能介绍

本工具可以帮助教师快速生成课程期初资料，包括：

教学大纲

生成专业的Word文档教学大纲，包含课程基本信息、教学目标、教学内容等

课程检查记录表

生成Excel格式的课程组期初教学资料检查情况记录表

审核汇总表

生成Excel格式的教学大纲审核汇总表

系统要求

  • Python 3.6+
  • 安装依赖：pip install -r requirements.txt

使用步骤

1准备JSON数据文件

您需要准备一个包含课程信息的JSON文件，格式如下：

[{
    "courseCode": "FIN0008B",
    "courseName": "财经公文写作",
    "teacherName": "任渝",
    "department": "智能金融学院",
    "credits": "2.0",
    "totalHours": "32",
    "courseNature": "专业拓展课",
    "applicableScope": "东区24级专资产评估与管理01班+证券实务01班",
    "englishName": "Course English Name"
}]

2准备模板文件

确保您的模板目录中包含以下文件：

  • 1 课程教学大纲基础模版.docx - 教学大纲模板
  • 附件3 课程组教学资料检查情况记录表-完全一致模板.xlsx - 课程检查记录表模板
  • 附件1 教学大纲的审核汇总表-完全一致模板.xlsx - 审核汇总表模板
  • 任渝.png - 课程组负责人电子签名（如果有）
  • 陈蔚.jpg - 开课单位负责人电子签名

3运行应用程序

在命令行中运行：

python course_materials_app.py

4使用应用程序界面

  1. 选择JSON文件：点击"浏览"按钮选择您的课程信息JSON文件
  2. 选择模板目录：点击"浏览"按钮选择包含模板文件的目录
  3. 选择输出目录：点击"浏览"按钮选择生成文件的保存位置
  4. 点击"生成课程资料"：程序会自动生成所有课程资料

生成结果

生成的文件将保存在您选择的输出目录中，按教师姓名和课程代码组织：

│ └── 智能金融学院-教学大纲审核汇总表-任渝.xlsx

├── 2026春-大数据分析基础-教学大纲-任渝.docx
├── 大数据分析基础-课程组期初教学资料检查情况记录表-任渝.xlsx
└── 智能金融学院-教学大纲审核汇总表-任渝.xlsx

注意事项

  • 确保JSON文件格式正确，包含所有必要字段
  • 确保模板文件存在且格式正确
  • 电子签名图片必须是PNG或JPG格式
  • 生成过程可能需要一些时间，请耐心等待

故障排除

  • 如果出现"文件不存在"错误，请检查文件路径是否正确
  • 如果出现"导入错误"，请确保已安装所有依赖
  • 如果生成的文件格式不正确，请检查模板文件是否完整
